# JogoDaVelha
Um jogo da velha para passar seu tempo se divertindo com uma IA que foi perfeitamente programada para te desafiar no jogo da velha.

# Uma Simples Introdução
Este foi um jogo da velha criado pelo Gemini e AI studio e com algumas modificações chegou a este nível de dificuldade, ele vai analisar todos seus movimentos para que ele consiga a vitória.
